BuildOrder Calculator (StarCraft) v1.01

The program uses genetic algorithms to find the fastest Build Order to reach a certain goal, like 6 Wraith, or dropping 20 Hydralisks in 5 Speed-Overlords.
It is somewhat like "Stratagema", except that you only have to enter the goals and let the computer do the dirty work ;-)
Please read the readme.txt before starting the program!!
Unfortunately I found already two new bugs ^^
- if you set "detailed info" to 0 the output is sometimes wrong
- the output file shows still "10:1" or "9:4" instead of "10:01" and "9:04"
- It is probably more exact when I add a second or two for moving a new produced scv/drone/probe to the mineral line
Well, these are minor bugs, but I'm working on it now.
An "1.01b" update should be out tomorrow!
Thanks for testing!

Army Calculator (VOR)

This is the first version of a Army-Calculator for the table-top game "VOR".
It is pretty useless if you don't play "VOR".
 The program itself uses genetic algorithms to find the best Union Counter-Force against an enemy army.
It *runs* and brings up a good solution though the processing speed is (relatively) slow and could be improved by factor ~2-4.
I'm working on it :)
Please read the read.me before starting the program!!
