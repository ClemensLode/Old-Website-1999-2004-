Server

Die Server, PII 350 System mit Betriebssystem Linux, werden von PureTec bereitgestellt. 

Die Verbindung ins Netz ist möglich durch: 

- 6x2 MBit zu Nacamar, Topnet, Cybernet und Medaways, 
- 1x155 MBit zu Topnet 
- 1x155 MBit zu Nacamar 
- 1x155 MBit zu Belwue.
