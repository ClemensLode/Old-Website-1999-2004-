Hu? The Claw? What's that!?

Well... it is a so called 'game controller', simply said a 'handy' keyboard.
It is no mouse, it stands still on 4 plastic feet.
If your fingers are not very short (well... 'The Claw' is about 19cm long) 'The Claw' will lie perfect in your hand.
The buttons have a good press-point so you normally will not hit a button accidently.
- Hu? The Claw? What's that!?
Technical details

- 'The Claw' has 10 Buttons and is connected to your computer via USB
- The Windows software supports 4 shift buttons so you have (4 + 1 (no shift)) * 6 = 30 Functions. Besides all possible key combinations you can even assign mouse clicks and delays.
- Under Linux 'The Claw' reveals additional power, thanks to a programmer who wrote a basic software which can be configured individually.
Currently I am writing on an Update which will support StarCraft much better than now.
- It weighs about one pound.
- What are the technical details?
Requirements

- Windows: probably a 'fast' Computer (>500 mhz) for a faster button response
- Linux: probably a 'slow' Computer and a Software Patch / Programming skills to adapt the Linux driver to your needs.
- Besides you just need an USB support :)
- What do I need for 'The Claw'?
Is 'The Claw' a cheat?

Is a very fast Internet connection in a 3d FPS game a cheat? Well... you gain an advantage over your enemy...
But i prefer an other point of view. You can type in your commands faster, you improve the handling of the game, something the programmers should have done in the first place.
A good example is (as always :) StarCraft. Though the key assignment is not bad, it could be much better.
- Is 'The Claw' a cheat?
The bad things about 'The Claw'

- You need some time to get around with 'The Claw'.
- 'The Claw' transmits its data fast, but sometimes (under windows) when using complex key combinations the software will forget some clicks... You have to insert a 100ms delay between the commands.
- It works for action and for RTS games, though very complex games (like StarCraft:Broodwar) need some tricky settings, as there are more commands than buttons.For example you need to put 'Build' and 'Barracks' on one button ("BB").
- there is no right-hand version
- The drawbacks
Why should I use 'The Claw'?

Well, there are some tools like DoxStar that works like 'The Claw', but your hand still lies much more comfortable on 'The Claw' as it can on the keyboard...
In addition optimizing the complex configuration possibilities are fun and quite useful in a game.
- Why should I use 'The Claw' instead a simple key-assign program?
Is it useful?

All in all:
I played a lot with the keyboard and the claw.
I did not win more games, neither I lost more with the claw, but it was more fun and it is better for the hand. :)
Besides it looks better than a standard keyboard and leaves more space on the table.
